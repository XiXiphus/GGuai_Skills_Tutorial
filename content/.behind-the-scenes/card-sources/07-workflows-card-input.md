# Card Input: 3 条最常用 Workflow

---

## Workflow 1: 学一个概念

**什么时候用**：遇到想懂的概念、术语、理论

**链路**：`ljg-learn` → `ljg-plain` → 可选 `ljg-card`

**你会拿到**：一段能讲给别人听的大白话理解

**一句话记忆**：先挖深，再讲清，想存就存

---

## Workflow 2: 吃掉一篇文章

**什么时候用**：读完/听完材料，想收下来

**链路**：`ljg-rank` → `ljg-learn` → `ljg-writes`

**你会拿到**：一段你能直接用的转述或观点

**一句话记忆**：先找骨架，再深挖，最后变成你的话

---

## Workflow 3: 做一张卡片

**什么时候用**：有想法了，想做成能发的成品

**链路**：`ljg-writes` → `ljg-card`

**你会拿到**：一张 PNG 卡片，可发朋友圈/存笔记

**一句话记忆**：先写清楚，再变好看

---

## 快速选择表

| 你现在的状态   | 走哪条          |
| -------------- | --------------- |
| 有个概念想懂   | Workflow 1      |
| 读完材料想收   | Workflow 2      |
| 有想法想做卡   | Workflow 3      |
| 不知道自己在哪 | 先用 `ljg-rank` |

---

## 提醒

这些是默认值，不是枷锁。熟了之后，你可以自己改、自己配。
