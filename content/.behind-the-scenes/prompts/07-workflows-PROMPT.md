# Module 7 Prompt: Workflows

请先读取：

- `../00-global-style.md`
- `../01-course-map.md`

然后在当前目录写出以下文件：

- `lesson.md`
- `talk-track.md`
- `exercise.md`
- `card-input.md`

## Module Goal

这一课要帮她完成一次升级：

从“我会调几个 skill”，升级到“我知道怎么把它们串起来”。

## What Must Land

她学完后要拥有至少三条能真复用的小工作流。

请至少设计这三条：

1. 学一个概念
2. 吃掉一篇文章或一份材料
3. 做一张可发出去的卡片

## Content Requirements

### `lesson.md`

- 先解释为什么单点 skill 好用，但 workflow 更稳。
- 每条 workflow 都要写清：
  - 触发场景
  - 第一步为什么这么起
  - 中间怎么切 skill
  - 最终产物是什么
- 优先使用这些 skill 组合：
  - `ljg-rank`
  - `ljg-plain`
  - `ljg-learn`
  - `ljg-writes`
  - `ljg-card`

### `talk-track.md`

- 写成我可以讲给她听的话。
- 听起来要像“给你三条好用的路”，不是“给你一套复杂系统”。
- 最好让她觉得：以后我遇事不用重新从零想了。

### `exercise.md`

- 给五个场景，让她把场景配给某条 workflow。
- 至少有一个题目需要她自己补上 workflow 中缺的一步。
- 最后让她试着设计一条属于自己的微工作流。

### `card-input.md`

- 输出一份“3 条最常用 workflow”的卡片文案源
- 要做到一眼看懂：什么场景 -> 调什么链路 -> 拿到什么成品

## Teaching Warning

- 不要把 workflow 画得太复杂
- 不要让她感觉自己要先学会全部 skill 才能开始用
- 她学完后应该有一种“我已经有路了”的松弛感
