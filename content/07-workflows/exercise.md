# 练习：配对工作流

## 第一部分：场景配对

下面有五个场景。请判断每个场景最适合用哪条 workflow，把场景编号填到 workflow 后面。

**Workflow 选项：**

- A. 学一个概念（learn → plain → 可选 card）
- B. 吃掉一篇文章（rank → learn → writes）
- C. 做一张卡片（writes → card）

**场景：**

1. 小红读了一篇关于"睡眠周期"的科普文章，觉得很有意思，想整理成自己能讲的东西。
2. 小明听到朋友提到"邓巴数"这个概念，不太懂，但想知道这到底是什么。
3. 小李读完一本书的一章，想把自己的收获做成一张朋友圈卡片发出去。
4. 小张看完一部纪录片，有很多感想，但还没组织成一段话，想先理清楚再做成视觉卡片。
5. 小王遇到一个陌生的术语"锚定效应"，想彻底搞懂它，还能讲给别人听。

**你的答案：**

- Workflow A（学概念）：场景 ___
- Workflow B（吃材料）：场景 ___
- Workflow C（做卡片）：场景 ___
