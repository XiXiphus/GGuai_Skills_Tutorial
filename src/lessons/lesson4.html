Lesson 04

Four Inputs · 因料施策

手里拿的东西不同，第一句话也不同。
先分清入口，再决定用什么刀。

同一件事，四种入口

假设你现在想了解「为什么人会拖延」。你可以从四个不同的地方进来：

入口一：File

你下载了一本《拖延心理学》的 PDF。

  • 不太好的开口方式：「你看一下这个文件」
  • 更好的开口方式：「这是一本关于拖延的书，我想先知道它最核心的三个观点是什么，不要细节」

为什么更好？因为你说清了你的位置：我刚拿到一本厚的，我需要先知道骨架，而不是被细节淹没。

入口二：URL

朋友发你一篇文章链接。

  • 不太好的开口方式：「总结一下这篇文章」
  • 更好的开口方式：「这篇文章讲拖延，我想知道作者反对什么常见说法，以及他提出的替代观点」

为什么更好？因为你给了 Agent 一个任务：不只是压缩内容，而是找出「对抗性」——作者在和什么较劲。

入口三：Paste

你在某个论坛看到一段写得很好的话，直接复制下来：

「拖延不是时间管理问题，而是情绪调节问题。我们拖延是因为任务引发的情绪不舒服，而不是因为任务本身难。」
  • 不太好的开口方式：「你怎么理解这段话」
  • 更好的开口方式：「这段话说拖延是情绪问题，我想知道：这个观点从哪里来？有什么证据或故事支撑？」

入口四：Concept

你没有材料，就是突然想搞懂「拖延」这件事。

  • 不太好的开口方式：「给我讲讲拖延」
  • 更好的开口方式：「我想理解拖延，但我不知道从哪里开始。你能先帮我列出这个主题下最常被误解的三个点吗？」

一条最有代表性的处理链

现在假设你手里有一篇长文（不管是 File、URL 还是 Paste），你想真正把它变成自己的理解。

这里有一条最稳的处理链：

Input
→
ljg-rank
→
ljg-plain
→
ljg-learn

第一步：ljg-rank（先抓骨架）

你对 Agent 说：

「这篇文章讲拖延，帮我先抓骨架：如果只能用三个核心机制解释为什么人拖延，是哪三个？」

为什么是这一步先？因为如果你一开始就让它「解释一下情绪调节理论」，你会陷入细节。先把骨架拎出来，你才知道整篇文章站在哪几块石头上。

注：这里的「抓骨架」就是 Lesson 03 说的「砍到几根硬支柱」——把复杂信息压到最不可再少的那几条。

第二步：ljg-plain（说人话）

现在你对 Agent 说：

「用 ljg-plain 把刚才的三个机制讲给我听，假设我在给朋友转述，不要用术语」

为什么这一步在中间？因为你刚刚拿到骨架，但骨架可能还带着术语。这一步是确保你真的能用自己的话讲出来，而不是背 definition。

第三步：ljg-learn（再挖一层）

最后你对 Agent 说：

「对第二点『情绪调节』用 ljg-learn，我想知道：这个说法是什么时候被发现的？它推翻了什么旧观点？」

为什么是这一步最后？因为如果你一开始就深挖，可能挖的是不重要的细节。现在你确认了骨架、确认了自己能讲出来，再选择最感兴趣的那个点往下挖，才是高效的。

顺序为什么不能反？

rank → plain → learn 的顺序，本质是让你先站稳，再往下跳。

Lesson 04 Drill

看菜下饭：不同材料怎么开口

场景判断练习

下面有四个场景，请判断：你怎么跟 Agent 开口？先调哪把 Skill？

1

场景一：你有一份长达 50 页的行业年度报告 PDF，你想快速了解这个行业今年最大的三个变化。

点击查看建议

输入方式：File 上传

第一步 Skill：ljg-rank

开口建议：「这是今年的行业报告，请先用 ljg-rank 帮我找出今年最大的三个变化，只说骨架，不要细节。」

2

场景二：你在书上看到一段话：「自由不是想做什么就做什么，而是不想做什么就可以不做什么。」你觉得很有道理，想深挖一下。

点击查看建议

输入方式：Paste 文本

第一步 Skill：ljg-learn

开口建议：「这段话说[粘贴文本]，请用 ljg-learn 帮我深挖一下，这个观点在哲学史上是谁先提出来的？它的反面是什么？」

3

场景三：朋友发给你一篇很长的公众号文章链接，标题很耸动，你想知道它到底在说什么，有没有逻辑漏洞。

点击查看建议

输入方式：URL 链接

第一步 Skill：ljg-rank

开口建议：「这篇文章讲[主题]，请用 ljg-rank 帮我拆解它的核心逻辑链，并指出它可能回避了什么问题。」

4

场景四：你突然想搞懂「量子纠缠」到底是什么意思，但你完全没有物理学 background。

点击查看建议

输入方式：直接提问（Concept）

第一步 Skill：ljg-plain

开口建议：「我想理解『量子纠缠』，请用 ljg-plain 给我解释，假设我是一个完全不懂物理的高中生，给我一个生活中的类比。」
