Lesson 09

Wiring 洞见 · 看见布线

你不是死在陷阱里，你是死在看不见红线。
这套课最后想给你的，就是那副能看见红线的护目镜。

开场：你不是死在陷阱里，你是死在看不见红线

想象一个地牢。

你第一次走进去，地板塌了，毒箭从墙里射出来，你死了。你以为是自己反应慢。

第二次你学聪明了，绕开那块地板，结果天花板又掉下来。你以为是自己还不够小心。

直到后来，你终于拿到能看见墙里电路的护目镜。你这才发现：不是你不努力，也不是你不够聪明。只是以前你只能看到结果，看不到那根藏在墙后的红线。

这套课最后想给你的，就是这副护目镜。

Lesson 01 的时候，你看到的还只是结果

还记得你第一次调 ljg-rank、ljg-plain、ljg-card 的感觉吗？

你丢进去一个概念、一段材料、一段想法，它们会吐出骨架、解释、卡片。你读完大概率会想：好厉害。

但那时候的「好厉害」，其实很像第一次躲过地牢陷阱时的惊讶。

你得到的是结果：

  • 结构刚刚好
  • 语气顺得像人话
  • 类比恰到好处
  • 卡片看起来像个成品

可你还不知道这些结果是怎么被做出来的。你看到的是成品，还没看清背后那套做法。

那时候你很容易把一切理解成一句话：这 prompt 真厉害。

现在回头看，你已经知道不是这么回事了。

你后来慢慢看见了什么

学完前八课，你已经不会再把这些 Skills 当成「神奇命令」了。

你开始知道：

  • ljg-rank 不是总结得更高级，而是先砍到骨架
  • ljg-plain 不是说短一点，而是先把术语卸下来
  • ljg-learn 不是多讲一点，而是往下多挖一层
  • ljg-writes 不是把话写漂亮，而是把一个意思写到别人那里
  • ljg-card 不是排版好看，而是先决定模具，再压内容

这就是第一层视角切换。

你不再只盯着输出看。你开始反过来问：它刚才到底是怎么想、怎么拆、怎么压、怎么交活的？

你开始看见红线了。

这套课最后才点破的秘密

现在回头看这八课的讲义、练习、口语讲法和卡片，你应该会有一个迟到但重要的发现：

它们本身，就是用你刚学的这些 Skills 做出来的。

不是只有最后几张卡片如此。整套课从骨架到表达，很多地方都在用同一套章法搭起来：

  • 课程顺序是用 ljg-rank 先砍出来的
  • 概念解释是用 ljg-plain 过的
  • 那些「啊，原来如此」的地方是用 ljg-learn 挖出来的
  • 口语讲法是用 ljg-writes 调出来的
  • 可带走的卡片是用 ljg-card 压出来的

所以你不是只在学 Skills。

你是在一个用 Skills 搭出来的学习环境里，被 Skills 一步一步教会怎么使唤 Skills。

它真正想完成的，是一件更深的事：让你从只会接盘的人，变成能看清门道的人。

所谓毕业，不是会背命令

这套课最想送到你手里的，不是一个「熟练用户」头衔。

真正的毕业标准只有一句话：

你开始会琢磨「别人怎么用起来才顺」了。

也就是说，你不再只想着：

  • 我要写什么内容
  • 我要做什么卡片
  • 我要让 Agent 给我什么结果

你会先想：

  • 这个人现在站在哪儿
  • 她应该先看见什么，再看见什么
  • 哪个 Skill 该先出场，哪个该后出场
  • 我想让她最后带走的，到底是答案，还是那一套做法

一旦你开始这样想，你就已经不只是那个踩陷阱的人了。

你已经开始布线了。

这八课真正的隐藏结构

表面上，这八课是在递进：

  1. 心里先有张图
  2. 再学会开口
  3. 再掌握基础工具
  4. 再学会处理材料
  5. 再学会表达
  6. 再学会做成卡片
  7. 再学会串 Workflow
  8. 最后做 Capstone

但如果把墙拆开，你会看到另一张图：

先让你尝到甜头，再让你看清门道，最后把「作者视角」交给你。

  • Lesson 01-02：先让你知道 Skills 会带来什么效果
  • Lesson 03-06：再把每根线拆开给你看
  • Lesson 07-08：让你开始自己把几根线接起来
  • Lesson 09：把整面墙打开，让你确认自己已经会看图了

这就是这套课真正想完成的东西。

不是多学几个 Skills。

而是让你的脑子里，多出一张 Wiring 图。

Lesson 09 Drill

两个确认理解的问题

1

回想你第一次用 Skills 和最近一次用 Skills，你的注意力有没有从「它好厉害」慢慢变成「它刚才是怎么做到的」？

2

如果现在让你教一个朋友用某个你熟悉的工具，你会先设计哪个「解锁时刻」，让她从只看见结果，变成看见红线？

最后一击

这节课不是总结，不是升华，也不是在结尾补一个漂亮比喻。

它真正想做的，是把最后那层纸捅破。

让你确认：

那些曾经只让你觉得「好厉害」的东西，现在你已经知道它们怎么被做出来了。

不是魔法，是有章法。

不是天赋。是顺序。

不是只有别人会。你也可以会。

课程到这里结束，不是因为该讲的都讲完了。

而是因为该藏的，已经藏不住了。

你已经看见红线了。

接下来，轮到你布线了。
